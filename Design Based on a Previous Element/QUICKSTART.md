## The 4-Step Speed-Optimized Process

### STEP 1: Analyze & Choose Framework (2-3 minutes)

**Quick element analysis:**
- What's the hero element? (character, landmark, animal, object)
- What makes it recognizable? (key visual features)
- What style? (cartoon, realistic, vintage, modern)
- What composition works best?

**Choose ONE compositional framework:**
1. **Centered Hero** - Element dominates center (60-80%)
2. **Environmental Integration** - Element in natural setting (35-40%)
3. **Diagonal Journey** - Path/trail composition with movement
4. **Radial Mandala** - Circular symmetry, concentric rings
5. **Narrative Scene** - Element performing activity
6. **Layered Depth** - Strong foreground/midground/background
7. **Frame-Within-Frame** - Decorative border frames element
8. **Asymmetric Balance** - Dynamic off-center energy
9. **Vertical Stacking** - Horizontal layers in portrait format
10. **Horizontal Spread** - Left-to-right panoramic flow

**Reference:** See COMPOSITION_FRAMEWORKS.md for details

### STEP 2: Select Elements (2 minutes)

**From regional libraries (when available), select 5-10 items:**
- **Flora:** 3-4 native plants
- **Fauna:** 1-2 supporting animals (if applicable)
- **Cultural:** 2-3 patterns/objects/architecture
- **Environmental:** Landscape features

**Time-saver:** Copy-paste from existing element libraries rather than researching from scratch.

### STEP 3: Write Prompt Using Template (3-4 minutes)

**Use PROMPT_TEMPLATE.md structure (200-350 words):**

1. **Format** (1 line) - Square 1:1 / Portrait 4:5 / etc.
2. **Subject** (1 sentence) - Hero + setting + destination
3. **Composition Structure** (2-3 sentences) - Framework + placement + flow
4. **Protagonist Element** (30-50 words) - Hero details, size %, position
5. **Supporting Elements** (40-60 words) - Simple bulleted list, 5-10 items
6. **Decorative Layer** (20-30 words) - Decoration density, pattern fills
7. **Text Integration** (20-30 words) - Primary 15-25% height, secondary 6-8%
8. **Style & Aesthetic** (30-40 words) - Copy from STYLE_DESCRIPTORS.md
9. **Color Palette** (15-20 words) - 4-6 simple color names (NO CMYK)
10. **Edge Treatment** (10-15 words) - Organic irregular default
11. **Production** (10-15 words) - One sentence at end
12. **"CREATE DESIGN"** - Required ending

**Copy-paste style descriptors from STYLE_DESCRIPTORS.md:**
- Whimsical Cartoon
- Realistic Illustrative
- Vintage Travel Poster
- Modern Flat Design
- Vintage Naturalist
- Decorative Folk Art
- Mystical Atmospheric
- And 13+ more options

### STEP 4: Generate & Iterate

Submit to AI image generator. If revision needed, adjust specific sections rather than rewriting entire prompt.

## Non-Negotiable Design Rules

### 1. Hero Element Size
**30-80% of design** depending on framework:
- Centered Hero: 60-80%
- Environmental Integration: 35-40%
- Narrative Scene: 35-45%
- Radial Mandala: 25-35%

### 2. Text Prominence (CRITICAL)
**Primary text: 15-25% of design height**
- Must be readable from 3 feet away
- Integrated into design, not floating
- Styled (hand-lettered, vintage, bold, decorative)
- Users WILL complain if too small

**Secondary text: 6-8% height**

### 3. Edge Treatment Default
**Organic irregular shapes** following natural element contours
- NO rectangles or straight edges (unless format requires)
- Border follows design elements naturally
- Die-cut sticker aesthetic

### 4. Uniqueness Requirement
**Each design for new client = must be unique**
- Never repeat compositions for same destination
- Change framework, style, or narrative angle
- Different supporting elements
- Variation in time of day, mood, activity

### 5. Celebratory Tone
- Inviting, festive, warm, welcoming
- NO cold, dark, sad, or sober tones
- Magical, cozy atmosphere
- Creative and interesting

## Common Mistakes to Avoid

❌ **Prompt too long** (>400 words)
→ Use template structure, trim descriptions

❌ **No composition framework chosen**
→ Pick one from COMPOSITION_FRAMEWORKS.md

❌ **Text too small** (<15% height)
→ Always specify 15-25% for primary text

❌ **Using CMYK colors**
→ Use simple color names: "bright turquoise" not "C60 M10 Y20"

❌ **Writing custom style essay**
→ Copy-paste from STYLE_DESCRIPTORS.md library

❌ **Supporting elements as paragraphs**
→ Use simple bulleted list

❌ **No decoration density specified**
→ State 6/10, 8/10, or 10/10

❌ **Forgot "CREATE DESIGN" at end**
→ Always include this

❌ **Repeating same composition for destination**
→ Use different framework for each new client

## Iteration & Revisions

**Common revision requests:**

**"Too childish"**
→ Change style to: Realistic Illustrative or Vintage Naturalist

**"Too busy/cluttered"**
→ Reduce decoration density from 10/10 to 6/10

**"Text too small"**
→ Increase primary text from current % to 20-25%

**"Wrong mood"**
→ Adjust color palette and style descriptor

**"Element too small"**
→ Increase hero element from current % to 45-50%

**Revision time:** 2-3 minutes with specific feedback
