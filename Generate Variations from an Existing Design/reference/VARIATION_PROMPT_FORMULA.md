## ‚ùå WHAT NOT TO INCLUDE

**NEVER include these in variation prompts:**
- CMYK or hex color codes
- Line weights or thickness measurements
- Material specifications
- Laser cutting details
- Production constraints
- Detailed positioning coordinates
- Font specifications
- Dimensions in points or pixels

**Instead use:**
- Simple color names ("turquoise ocean", "golden sunset")
- General size relationships ("giant", "tiny", "oversized")
- Creative descriptions ("vintage", "playful", "dramatic")

## üéØ KEY PRINCIPLES

### 1. CONCEPT LEADS
Start with the creative transformation idea - it guides everything else

### 2. ACTION VERBS DRIVE
Use active language:
- "Swimming through"
- "Holding camera"
- "Flying between"
- "Building puzzle"
- "Surfing on wave"

### 3. BREVITY IS POWER
Short, focused prompts generate better variations than verbose specifications

### 4. TRANSFORMATION OVER PRESERVATION
80% describing what changes, 20% context

## üöÄ QUICK GENERATION PROCESS

### Step 1: Define Concept (30 seconds)
"What if this design was inside a snow globe?"

### Step 2: Describe Spatial Change (1 minute)
How does the composition restructure for this concept?

### Step 3: Activate Characters (30 seconds)
What are they DOING in this new concept?

### Step 4: List New Elements (1 minute)
Quick brainstorm 5-10 items that fit concept

### Step 5: Summarize Changes (1 minute)
Before ‚Üí After for main transformations

**Total time: 4 minutes per variation**
