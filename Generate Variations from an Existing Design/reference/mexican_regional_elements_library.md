## SONORA Region

### Flora

**Saguaro Cactus** (Carnegiea gigantea)
- Visual: Tall columnar cactus with upward-reaching arms, ribbed texture, white blooms at top
- Cultural: Iconic symbol of Sonoran Desert, can live 200+ years
- Colors: Deep green body (CMYK: 50, 0, 100, 30), white flowers (CMYK: 0, 0, 0, 0)
- Laser-cut: Simple columnar silhouette, 1.5pt minimum line weight, arms connect to body

**Palo Verde Tree** (Parkinsonia spp.)
- Visual: Green-barked branches with tiny leaflets, bright yellow spring blooms
- Cultural: Arizona/Sonora state tree, name means "green stick"
- Colors: Lime green bark (CMYK: 40, 0, 80, 0), yellow flowers (CMYK: 0, 10, 100, 0)
- Laser-cut: Branching silhouette with delicate leaf details for UV print

**Ocotillo** (Fouquieria splendens)
- Visual: Tall spiny stalks radiating from base, red tubular flowers at tips
- Cultural: Desert adaptation symbol, appears dead until rain
- Colors: Gray-brown stalks (CMYK: 0, 10, 30, 50), red blooms (CMYK: 0, 100, 80, 0)
- Laser-cut: Radiating straight lines with small flower details

**Agave** (Agave americana)
- Visual: Rosette of thick pointed leaves, some varieties blue-gray
- Cultural: Source of tequila/mezcal, sacred to indigenous peoples
- Colors: Blue-gray (CMYK: 30, 10, 30, 0) or green (CMYK: 50, 0, 80, 20)
- Laser-cut: Symmetrical pointed leaf arrangement

**Prickly Pear/Nopal** (Opuntia spp.)
- Visual: Flat paddle-shaped segments, colorful fruit (tunas)
- Cultural: Mexican national symbol, important food source
- Colors: Green pads (CMYK: 40, 0, 100, 10), magenta fruit (CMYK: 10, 100, 0, 0)
- Laser-cut: Overlapping paddle shapes, spines as printed detail

### Fauna

**Roadrunner** (Geococcyx californianus)
- Visual: Long tail, distinctive crest, ground-running bird
- Cultural: Desert speed symbol, state bird of New Mexico
- Colors: Brown/tan (CMYK: 0, 40, 60, 40), blue/red accents
- Laser-cut: Dynamic running pose, 1.5pt line weight minimum

**Bighorn Sheep** (Ovis canadensis)
- Visual: Curved spiral horns, muscular body, mountain dweller
- Cultural: Symbol of desert wilderness, endangered species
- Colors: Tan/brown (CMYK: 0, 30, 50, 20)
- Laser-cut: Silhouette showing distinctive horn profile

**White-winged Dove** (Zenaida asiatica)
- Visual: Distinctive white wing patches, cooing call
- Cultural: Sonoran summer symbol, important to Tohono O'odham
- Colors: Gray-brown (CMYK: 0, 20, 30, 40), white wing bars
- Laser-cut: Flight or perched silhouette with wing detail

**Javelina/Peccary** (Pecari tajacu)
- Visual: Pig-like, bristled coat, travels in groups
- Cultural: Desert forager, featured in O'odham stories
- Colors: Dark gray-brown (CMYK: 0, 30, 40, 60)
- Laser-cut: Simple side profile or group silhouette

**Gila Monster** (Heloderma suspectum)
- Visual: Beaded orange/black pattern, venomous lizard
- Cultural: Protected species, indigenous medicine symbol
- Colors: Black and orange pattern (CMYK: 0, 70, 100, 40 / 0, 0, 0, 100)
- Laser-cut: Distinctive beaded pattern as print detail

## Usage Tips

**Combining Elements**:
- Use 3-5 elements per design for visual balance
- Mix categories (flora + fauna + cultural motif)
- Ensure regional accuracy (don't mix Sonoran + Yucatán unless intentional)

**Adapting for Laser-Cut**:
- All elements listed with minimum 1.5pt line weights
- Complex patterns noted as "print detail" vs. "cut detail"
- Structural connections required for all cut elements

**Color Coordination**:
- CMYK values provided as starting points
- Adjust saturation for cohesive palette
- Consider wood base color in final color choices

**Cultural Sensitivity**:
- Sacred symbols (Maya glyphs, Yaqui ceremonies) require respectful treatment
- Research specific community permissions for commercial use
- Avoid stereotyping or appropriation

**Scaling for Format**:
- Small formats (keychain, magnet): Use 1-3 simple elements
- Medium formats (ornament, coaster): Use 3-5 elements
- Large formats (wall art, poster): Use 5-7+ elements with detail
